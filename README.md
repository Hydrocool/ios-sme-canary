Throwaway canary app for **Slice 1** of the iOS SME builder. A one-screen SwiftUI tip
calculator, built by the **Sierra** persona to prove it can drive the Slice-0 cloud-macOS
verify loop end-to-end (grill → plan → test-driven-development → push → read CI verdict →
green). Reuses the Slice-0-proven `ios-ci.yml` workflow verbatim.

Public, $0 (free macOS Actions minutes). Not a real product.
